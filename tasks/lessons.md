# Lessons

- 2026-04-23: When updating repo docs, keep installation guidance near the top of `README.md` so the first-use path is visible without scrolling.
- 2026-04-23: For cc-bridge hook UX, do not collapse scope into a boolean flag. Model user scope from `~/.claude/` and project scope from the current repo or an explicit project path, then force the selection flow before any hook write.
- 2026-04-23: When answering README prerequisite questions, sweep runtime commands, source-file assumptions, activation steps, and workflow secrets together instead of stopping at the first missing dependency.
